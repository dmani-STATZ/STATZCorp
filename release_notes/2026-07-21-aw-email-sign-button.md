The **Sign** button in Arctic Wolf security awareness emails appeared as a blue
box with no label and no link when the message was opened or composed in
Outlook. The button now renders and links correctly in Outlook, Outlook on the
web, Gmail, and Apple Mail.

A plain-text version of the completion link now appears beneath the button, so
the training page stays reachable even if a mail client blocks the button.

The email preview page and the downloadable `.eml` file are now generated from
the same source, so what you see in the preview is exactly what recipients
receive.
