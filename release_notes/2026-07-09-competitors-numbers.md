The **Competitors Numbers** page lets the team maintain a shared watchlist of competitor CAGE codes and see how many DIBBS awards each has won this week, month, quarter, and year — including dollar totals.

Open it from the Contract Dashboard or **Contract Menu → Competitors Numbers** (`/sales/competitors/`). Every logged-in user can view the full list and add a CAGE code. Only the person who added an entry can remove it; others see a disabled Remove button with the adder's name shown on each card.

Company names come from the existing SAM.gov cache (seeded when you add a CAGE). Award statistics use nightly-scraped `DibbsAward` data (non-faux rows only). A banner at the top shows the earliest tracked award date so you know how far back the numbers go.
