The PO Acknowledgment Letter now opens on a dedicated full page instead of a modal on Contract Management. A live HTML preview pane lets you refresh the letter layout as you edit fields without saving first.

PO Acknowledgment Letter preview now renders directly from the uploaded ISO template. Uploading a new template revision immediately updates the preview.

Staff can upload a new Word letter template with drag-and-drop directly on the letter page; the new revision is activated automatically—no server file access required.

The **Generate .docx** button works the same as before: it saves your fields and downloads the filled document from the active template.
