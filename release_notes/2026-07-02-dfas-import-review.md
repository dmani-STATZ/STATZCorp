The DFAS Payment Import review screen now lets you apply payments on your own schedule. Each matched row has its own checkbox and **Apply Now** button, so you can import one payment or bulk-apply any subset. **Apply All Matched** still applies every ready row in one step. Applying rows no longer closes the batch — use the separate **Close Batch** action when you are finished (you will be prompted if unresolved rows remain). Already-matched rows can be **Re-matched** to change the contract, CLIN, or shipment. Click the preview icon next to **Resolved To** to see match details and open the contract record in a new tab before you apply.
