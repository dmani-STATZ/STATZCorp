## Finance Audit — CLIN Item # links to detail page

On the Finance Audit page, each CLIN **Item #** in the CLINs table is now a link to that CLIN’s detail page. You can open split values and other CLIN-level data directly without leaving the finance workflow. The shipment-mode **S** badge still appears next to the link when applicable.
