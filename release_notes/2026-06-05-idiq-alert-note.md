## IDIQ Alert Note

A new **Alert Note** field is now available on IDIQ contracts.

When text is saved in the Alert Note field, a notice popup is automatically displayed
to analysts the moment they match that IDIQ on a Processing or Intake contract —
ensuring important context (such as "Packagings already listed and quoted") is never
missed.

The Alert Note can be set or updated from:
- The IDIQ contract edit page (Contracts app)
- The IDIQ processing editor (Processing app)
- The IDIQ draft editor (Intake app — IDIQ type drafts only)
