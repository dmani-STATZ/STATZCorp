DIBBS-sourced drafts in the Intake queue now have a **Fetch PDF** button. For any
skeleton draft created from the nightly awards scraper, clicking Fetch PDF will
automatically download the award document from DIBBS, parse it, fill in the draft
with the full contract data (CLINs, packaging, pricing, etc.), and upload the PDF
to the SharePoint folder  all with a single click.

Drafts that have already been parsed or manually edited are not affected.
