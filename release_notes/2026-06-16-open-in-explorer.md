Contract document folders can now be opened directly in **Windows Explorer** from the document browser **Actions** menu.

Choose **Open Current Folder in Explorer** to jump to the folder you are browsing, or select a single subfolder and use **Open Selected Folder in Explorer**. The action appears only when the folder maps to your locally synced OneDrive library.

Requires the STATZ desktop handler (`statzfile://` protocol) deployed to your workstation via IT.
