## Finance Audit — Plan Gross is now direct-edit

Plan Gross on the Finance Audit page is now edited by clicking the value and typing the number directly, instead of logging a payment-history entry. This matches the direct-edit flow for Item Value and Quote Value, and every change is fully recorded in the field's history.
