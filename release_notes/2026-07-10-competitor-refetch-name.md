On **Competitors Numbers**, watchlist rows that show **name unavailable** now include a **Refetch name** action. Clicking it forces a fresh SAM.gov lookup for that CAGE (bypassing the 30-day cache) so a previously failed lookup can resolve after the underlying issue is fixed — for example an expired API key.
