**Finance Audit — Shipment naming + simplified Add Shipment flow**

Shipments can now be given custom names (e.g. "Shipment Derek", "Shipment Nancy") instead of the generic Shipment 1, Shipment 2 numbering. Names can be set when creating a shipment or by clicking the name inline on the Finance Audit page to rename it. A single **Add Shipment** button in the CLINs header replaces the per-CLIN add pills — select your CLIN from the dropdown and fill in the details from one place.
