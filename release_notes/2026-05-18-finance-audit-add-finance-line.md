**Finance Audit — Add Finance Line in one step**

Adding a finance line no longer requires two separate entries. The new **+ Add Finance Line** button in the CLINs header opens a single modal where you select the CLIN, optionally scope it to a shipment, enter the line type, amount, and date — and the payment is logged automatically at the same time. Amount Paid defaults to match Amount Billed so the common case (fully paid at entry) requires no extra clicks. Set Amount Paid to $0 to save an unpaid line for later.
