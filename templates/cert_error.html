Security Certificate Setup Required

To ensure a secure connection, you need to install our security certificate.

Step 1: Download the Certificate

Download Certificate

Step 2: Install the Certificate

  1. Double-click the downloaded certificate file
  2. Click "Install Certificate"
  3. Select "Local Machine"
  4. Click "Next"
  5. Select "Place all certificates in the following store"
  6. Click "Browse" and select "Trusted Root Certification Authorities"
  7. Click "Next" and then "Finish"

Step 3: Clear Your Browser Data

To ensure the changes take effect:

  1. Close all browser windows
  2. Open a new browser window
  3. Clear your browser cache (Ctrl+Shift+Delete)
  4. Select "Cached images and files"
  5. Click "Clear data"
